### Валидация моделей на cloud.ru


## Папка embeddings (результаты валидации модели bge-m3):

testEmbeddings.http - файл тестовых запросов.
Файл для расширения REST Client в VS Code. Содержит 5 подготовленных HTTP-запросов к API эмбеддингов модели bge-m3 на cloud.ru.

testsResults.md — отчёт о результатах.
Сводный документ с метриками, наблюдениями и выводами по каждому из 5 тестов.

Подпапка testsResponses - архив полных JSON-ответов от cloud.ru
