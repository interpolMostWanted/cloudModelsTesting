## Отчёт: Валидация модели эмбеддингов bge-m3 на cloud.ru

### Конфигурация теста

| Параметр  | Значение |
| ------------- | ------------- |
| Провайдер  | cloud.ru (Model Run)  |
| Модель  | BAAI/bge-m3 (внутреннее имя: model-run-seeq5-alone)  |
| Base URL  | https://a4a09eb3-5d94-453b-b136-43a96eebb657.modelrun.inference.cloud.ru  |

| Формат запроса  | ({"model": "...", "input": "..."})  |

| № | Название | Запрос | Цель теста |

| 4 | test_synonyms | "Где взять логин для входа в личный кабинет?" | Семантическая близость к Тесту 1 (синонимы: «войти» ≈ «логин») |

### Итоговые выводы

1. Базовая совместимость

Все запросы возвращают 200 OK
Формат ответа полностью совместим с OpenAI API (data[0].embedding, usage, model)
Размерность вектора стабильно 1024 — соответствует спецификации bge-m3

2. Обработка длины текста

Токены: 11 → 21 → 28
Время: ~1.2с → ~1.5с → ~1.8с

Линейный рост времени обработки — признак здоровой работы модели. Ошибок 400 или 413 не зафиксировано.

3. Семантическое качество (ключевой критерий для RAG)

| Критерий | Результат |
| ------- | ------- |
| Понимание синонимов | ✅ «войти» ≈ «логин» — векторы близки |
| Различение тем | ✅ «ГАС» ≠ «принтер» — векторы удалены |
| Учёт контекста | ✅ Развёрнутый вопрос даёт иной вектор, чем короткий |
